# MoMo Budget App 💛

A personal finance tracker built for Ghana's MTN MoMo ecosystem. Track your daily income and expenses by category, visualize spending breakdowns, and get weekly saving tips.

## Live Demo
Coming soon (deploying to Vercel)

## Figma Design
[View Design Prototype](https://www.figma.com/make/B6l5KIUYdp4UqDpN3i7HZ7/Mobile-Budget-App-Screens)

## Features
- Log income and expenses by MoMo category
- View spending breakdown with donut chart
- Weekly saving tips tailored for Ghana
- Data saved locally in your browser

## Tech Stack
- React + TypeScript
- Vite
- CSS Variables (custom design tokens)
- Recharts (charts)
- React Router (navigation)

## Run Locally
```bash
npm install
npm run dev
```
